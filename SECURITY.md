# Security Policy

> ⚠️ **MedScribe-R-Us is a fictional company.** This repository is a portfolio
> project for educational purposes. There is no real product, no real users, and
> no real vulnerabilities to report.

---

## Reporting Vulnerabilities

If you believe you've found a security issue **in this repository itself**
(e.g., a misconfigured GitHub Actions workflow, an accidentally committed secret,
or a security flaw in the example code), please open a GitHub Issue or reach out
directly:

**Contact:** m.delrio0112@gmail.com  
**Subject line:** `[MedScribe-R-Us Portfolio] Security Issue`

There is no bug bounty associated with this repository. Issues will be addressed
as time allows.

---

## Purpose of This Repository

This repo exists to demonstrate AppSec program-building skills for portfolio purposes.
All company details, patient data, system names, and configuration are **entirely
fictional**.
